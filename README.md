## 프로젝트 개요

---

해야 할 일과 가야할 장소를 관리하는 모바일 to do list 어플리케이션입니다.<br> React-Native 개발의 기초를 익히기 위한 미니 프로젝트입니다.<br>
Work탭과 Travel탭으로 구분되며 각각 해야할 일, 가야 할 장소를 직접 작성해 추가할 수 있습니다.<br>
할 일을 수정하거나 목록에서 제거할 수 있습니다.
